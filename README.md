# Dragonballz
An open world turn-based fight game.

Features:
 - Character creation
 - Save Progress
 - Load Progress
 - Switch Character
 - Fight
 - Upgrade
 
Fight:
 - Phyiscal Attack
 - Super Attach 
 - Ultimate Attack
 - Block
 - Eat Senzu Bean
